# my-first-vue
第一次建立vue(vscode终端)



#### 第一种方式：使用Vue脚手架进行模块化开发

1.npm install -g @vue/cli   安装脚手架全局配置

2.vue create 项目名   

3.启动项目 npm run serve

## 第二种使用webpack init

1.全局安装脚手架

​	npm i -g @vue/cli-init 

2.创建项目

​	vue init webpack 项目名
